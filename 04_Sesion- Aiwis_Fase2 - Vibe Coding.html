Metodología de Desarrollo en 4 Pasos

1

Contexto (System)

Definir el rol del agente en AI Studio (System Instructions). Establece las reglas del juego.

2

Requerimientos (PRD)

Especificar requerimientos técnicos, agrupar múltiples necesidades en un solo prompt estructurado.

3

Generación (Build)

Ejecución del prompt para generar prototipos interactivos. Configurar Temperatura y Tokens.

4

Ajustes (Iteración)

Marcha blanca. Refinar mediante conversación continua sin rehacer todo el trabajo desde cero.

Aplicaciones Móviles (PWA)

No es necesario programar en Android Studio nativo desde el día 1. Las PWA (Progressive Web Apps) permiten "instalar" la web en el dispositivo Android con un icono directo, funcionando como aplicación nativa para casos de uso personales o prototipos rápidos.

Mencionado por Aiwis para el proyecto de Loreto

Base de Datos & Google Auth

Guardar datos en la caché del navegador no es escalable para apps reales. La integración fundamental mencionada es el uso de Firebase con autenticación (Auth). Esto permite gestionar usuarios multiusuario y persistir la información de manera segura.
